SR2400 Smart IoT Reader

Description:

ST2400 is a Smart IoT Reader which can support our smart tags to achieve different information collection. It works well with our 2.4GHz Smart Tags (ST) series such as

ST2401 :

It is a basic and simple tag. It will be continuously operating once battery is installed. RF signal (ID code) is transmitted every second. Battery standby life time is about one year. It can communicate with Smart IoT Reader SR2400 for identifying the location of any item. It uses button cell CR2032 which can be replaceable by customer.

ST2402 :

It is a visual and audible tag. It will be continuously operating once battery is installed. RF signal (ID code) is transmitted every second. It is built-in with two Red LED, two Green LED and a Buzzer. Customer can use the PC demo application to remotely control the flashing of LEDs and buzzer independently. Hence, it is super easy to identify the location of any item which is attached with this tag. Battery standby lifetime is about one year. It can communicate with Smart IoT Reader SR2400 for identifying the location of any item. It uses button cell CR2032 which can be replaceable by customer.

ST2403 :

It is a Motion Sensor Tag. It will be triggered to transmit RF signal (ID code), once per second, when it is suffered a very small movement. It will automatically stop RF Transmission after a period of about 10 minutes without movement. It can communicate with Smart IoT Reader SR2400 for identifying whether any object is at rest or in motion. It uses button cell CR2032 which can be replaceable by customer.

ST2404 :

It is a Light Sensor Tag. It is built-in with two Red LED and two Green LED as well . Customer can use the PC demo application to remotely control the flashing of these LEDs. It will stop RF signal transmission if the ambient is completely dark for about 5 minutes. It will automatically wake up to start RF transmission, once per second, when the ambient brightness is increased for about 1 minute. It can communicate with Smart IoT Reader SR2400 to identifying whether any object is at dark region or not. It uses button cell CR2032 which can be replaceable by customer.

Demo Application:

We will supply customer with the Smart IoT Demo Application execution file as well as the Visual C source code in which customer can further customize the application accordingly. The Demo Application presents in real time of below information :

The Smart Tag Identification Number (ID)

The distance (in cm based on received signal level dBm ) of any Smart Tags within its proximity

The battery level status (Normal / Low-Batt)

The battery level value (in data field : a0 = 100% full )

The duration (count in second) of Smart Tags within its proximity

If any Smart Tag goes out of range, it will be marked in red

The Demo Application allows customer to enter the Tag No. so that it can be searched or located easily with Green or/and Red LED or together with generating beep sound !!!
