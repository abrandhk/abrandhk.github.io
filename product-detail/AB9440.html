AB9440 RFID Reader

Description:

AB9440 RFID Reader Module Development Kit support ISO1800-6C/EPC C1G2 Protocols. It is connected with Raspberry Pi 3 single-board computer for system control and external connection. User can easily upgrade to Rasberry Pi 4 if necessary. Furthermore, it has built-in provision for connecting with Arduino and HC-05/HC-06 as well. User can easily enable this feature by setting the internal jumper connection. For the GPIO port interface, it provides 2 x GPI (input) , 2 x GPO(output) and +5V DC. User can easily control the logic level or red the external logic state through firmware command. It has built-in buzzer which will generate beep tone when power up initialization is done and during reading tag. Moreover, there is one low level mini-USB interface port for connecting the RF module directly to your PC or laptop. User can then run the PC Demo Application accordingly. Raspberry Pi Demo Application (Python code version) and PC computer Demo Application (CSharp code version) are provided so that user can further develop their smart reader features and operations according to different requirements. Sources codes are provided. We highly recommend to use our 8dBi circular polarization patch antenna to achieve about 9 ~12 meter read range.
