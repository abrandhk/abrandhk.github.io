ABT83-white Active Tag

Description:

ABT83-white RFID Active Simple Tag is equipped with built-in battery as its power source. User can replace the battery (CR2032). It works together with our RFID Simple Reader models AB305 and AB305X. This active tag can continuously operating for 2 years. When the tag battery level drop below the threshold voltage, it will automatically send out warning message to alert the reader. By design, it will wake up and transmit its ID code 3 times every second. The tag identification code ( Tag ID) is printed on the chassis. Since the duration of the transmitted pulse is very short, the power consumption is extremely low. When a reader receives the transmitted signal from RFID active tag, Reader will decode and upload the data to Host PC for system application usage.
