ST2403 Smart IoT Motion-sensor tag

Description:

It is a Motion Sensor Tag. It will be triggered to transmit RF signal (ID code), once per second, when it is suffered a very small movement. It will automatically stop RF Transmission after a period of about 10 minutes without movement. It can communicate with Smart IoT Reader SR2400 for identifying whether any object is at rest or in motion. It uses button cell CR2032 which can be replaceable by customer.

Demo Application:

We will supply customer with the Smart IoT Demo Application execution file as well as the Visual C source code in which customer can further customize the application accordingly. The Demo Application presents in real time of below information :

The Smart Tag Identification Number (ID)

The distance (in cm based on received signal level dBm ) of any Smart Tags within its proximity

The battery level status (Normal / Low-Batt)

The battery level value (in data field : a0 = 100% full )

The duration (count in second) of Smart Tags within its proximity

If any Smart Tag goes out of range, it will be marked in red

The Demo Application allows customer to enter the Tag No. so that it can be searched or located easily with Green or/and Red LED or together with generating beep sound !!!
