AB9301 RFID Reader

Description:

AB9301 is a high sensitivity and compact size RFID Reader module which supports ISO1800-6C/EPC C1G2 Protocols. The maximum output power is +30dBm. It has built-in dedicated algorithm to improve the yield for tag identification. It also uses high quality PCB substrate to minimize the path loss. Furthermore, it uses high quality material to achieve optimal shielding performance! It works well with our 8dBi circular polarization antenna ( URL: https://www.ebay.com/itm/362755599464 ) to achieve over 10 meters read range if using high sensitivity RFID dog-bone tags. AB9301 can communicate with outside host via the UART interface. It uses a standard connector. In addition, User can also customize the GPIO ports to control different devices for various applications. User can connect the AB9301 to their main board (host) through the standard Molex connector (53261-1571) , can be purchased from our eBay store, to easily customize any PCBA interconnection. Moreover, full set of API library is provided so that customers can easily further develop for different application beside our reference demo application provided.
