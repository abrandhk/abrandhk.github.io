AB9300 RFID Reader

Description:

AB9300 RFID Reader Module Development Kit support ISO1800-6C/EPC C1G2 Protocols. It can achieve 15 meters stable read range if using our 8dBi circular polarization patch antenna together with the Monza R6 Dogbone tag. Roger PCB substrate is used to achieve high performance in read rate , read range and high sensitivity. RS-232 interface board is included so that user can easily connected to Host PC with normal USB-to-RS-232 cable which included in the listing. DemoApp application is provided so that user can perform read tags operations after connecting the RS-232 port and antenna (NOT Include in this listing, Users can refer to our 8dBi circular polarization antenna listing item in our store). We highly recommend to work with our 8dBi circular polarization patch antenna to achieve about 15 meters read range. It supports wide range of input voltage (DC9V ~ 25V) . Furthermore, Demo Application source code is provided for customer further development. Firmware commands sets are also provided for user to develop and customized their application.
