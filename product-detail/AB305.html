AB305 2.4GHz Active Reader

Description:

AB305 is a professional long-range active RFID reader design which dedicated to read our long-range active tags. After reading the active tagsâ€™ ID , the reader will upload to the Host PC through the USB port. The Host can initiate different actions such as open-door, close-door or alerting accordingly. AB305 long-range RFID reader can achieve 20m read range together with the long-range active tag included in this listing. If you use 14 dBi antenna, the read-range can be extended to about 50 meters. It has built-in RF signal attenuator which can control the effective read range per your need. The accuracy of the read range adjustment can be about 1 meter. This long-range active RFID tag reader AB305 works well with our long-range active Tags models such as ABT83-black and ABT83-white . These Active Tags have built-in battery. Tag ID signal will be broadcast every second. The maximum life-time can achieve as long as 4 years if using active tag AB81. You can easily create your custom application based on our open source DemoApp source code. The communication protocol and frame format are simple which have been fully explained within the Product User Manual.
