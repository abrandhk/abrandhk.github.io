ST2409 Smart IoT Normal Temperature-sensor Reader kit

Description:

Smart IoT Temperature Tracker is a unique system which include a Smart IoT Reader SR2400 , Smart Temperature Tags (ST2405) and dedicated Window Application. It is design for user who want to track the temperature status at multiple locations with high precision. Two pieces of Smart Temperature tags have been included in the listing. We use high precision integrated-circuit temperature sensor from Texas Instrument (TI) which has been calibrated in manufacturing.

Smart IoT Temperature Tracker PC Application Description:

The PC Application presents in real time for below information in different columns :

[Item]

Showing how many Smart IoT Temperature Tags have been detected by the Smart Reader

[Device ID]

Showing the identification code which are unique to each Smart IoT Temperature Tags.

[Battery]

Showing the battery level status (either “Normal” or “Low-Batt” ). If it shows “Low-Batt”, please replace the button cell with a new CR2032

[Temperature / Degree]

Showing the real time measured ambient temperature from the Smart Temperature Tags

[Distance /m]

Showing the estimated distance of Smart IoT Temperature Tag with the Smart Reader station based on the received signal strength.

[Counting]

Showing that the Smart IoT Temperature tags are in range if the counting up is continuously
